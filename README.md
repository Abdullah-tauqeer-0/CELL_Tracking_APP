# Cell Tracking App

![Version](https://img.shields.io/badge/version-0.5.0-blue)

A modular Python application for tracking cells in microscopy videos using Kalman Filters.

## Features
- **Tracking:** Kalman Filter + Hungarian Algorithm
- **GUI:** User-friendly interface built with PyQt5
- **Analysis:** Velocity and displacement metrics

## Installation
```bash
pip install -r requirements.txt
```
